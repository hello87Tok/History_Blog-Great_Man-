# 概要
「Great Mans」は歴史上の記事を多数のユーザーが投稿しあえるブログプラットフォームです。
![pic2](https://user-images.githubusercontent.com/54434253/66112493-f455cf00-e605-11e9-8444-ed4d06bd1efe.png)

# 機能
* 記事一覧表示(will_paginate)
* 記事投稿
* カテゴリー追加
* カテゴリー一覧
* 一般ユーザー/管理ユーザー登録
* 管理ユーザーログイン
* 管理ユーザーのみのアクション(ユーザー削除、カテゴリー作成、全記事への編集権限)
* ユーザー一覧
* ユーザーアイコンアップロード機能(fog-aws/carrierwave/dotenv-rails)

# 環境構築
* windows10 Vagrant centos7

# 技術
* Rails5
* Github
* Heroku
* AWS S3/IAM

# 工夫点
1. ユーザーを一覧で表示したときに、画像を設定していないユーザーだけ一般的なアイコンを表示
2. .envファイルを作成して、セキュリティに配慮
